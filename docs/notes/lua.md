## Lua的小特性：集合
虽然Lua里面把它自己的数组称作`table`，但是我认为这个数据结构更类似于Python的集合或者json，里面可以存储任何东西，不受结构约束。
```lua
collection = {{"a", 1, 2}, "abc", {2, 3}}
```
如果需要查询数组的长度，只需要在数组名前面添加`#`即可得到
```lua
list = {1, 2, 3, 4, 5}
for i=1,#list do
    print(list[i])
end
```
Lua的数组（集合）下标从`1`开始，不同于C类语言的`0`

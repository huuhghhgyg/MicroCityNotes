1.1 MicroCity 的功能

MicroCity Web 的目标是提供一个简单快速的空间建模工具，可以随处运行。

数据分析

借助可编程脚本和丰富的 2D/3D 场景库，MicroCity Web 可用于分析并生动显示各种数据。

模型构建

借助内置的脚本调试功能和混合整数规划库，MicroCity Web 可用于创建物流模型并微调其参数。

仿真开发

感谢离散事件仿真库，MicroCity Web 可以动态评估各种物流情景并以动画方式显示出来。

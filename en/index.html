Getting Started with MicroCity Web

  1. Open MicroCity Web, and the example code for rotating blocks will load by default.

  2. Click the button to run the code.

  3. The program will run, and you will see the blocks rotating in the right-hand view.

If you want to learn more about how to use MicroCity Web and its advanced features, you can refer to the Learning Notes provided on this site.

Getting Started with MicroCity Desktop

  1. Go to the MicroCity documentation repository to download the Latest Version of MicroCity.

  2. Open the ScriptEditor and type:

Print("hello, world!")

  1. Save the script as test.mcs and drag it into MicroCity.

  2. In the main interface of the program, go to the Modules tab on the left side, open test, and double-click on main to execute the script you just wrote.

  3. The information about the execution of the script will be displayed in the Messages panel at the bottom of the program interface.

For more information on how to use MicroCity, you can refer to the documentation found on this site, which is a mirror of the documentation provided on GitHub: MicroCity Documentation. You can also refer to the documentation of MicroCtiy on GitHub at MicroCity.

开始使用MicroCity Web

  1. 打开MicroCity Web，默认加载方块旋转的示例代码。

  2. 点击运行按钮运行代码

  3. 程序运行，看到右侧视图中方块旋转。

更多MicroCity的使用方法可以参考本站翻译的 MicroCity Web中文文档。

如果想要深入了解 MicroCity Web 的用法和使用技巧，可以参考本站提供的 学习笔记

还可以参考老师的教材，介绍了很多算法

开始使用MicroCity Desktop

  1. 转到MicroCity的文档仓库下载 MicroCity的最新版本

  2. 打开 ScriptEditor 并输入:

Print("hello, world!")

  1. 将脚本另存为 test.mcs 并将其拖入 MicroCity

  2. 在程序主界面左侧转到 Modules 标签页，打开 test ，双击其中的 main ，执行刚刚编写的脚本。

  3. 程序界面下方的 Messages 栏中会输出执行脚本的信息。

更多MicroCity的使用方法可以参考本站搬运自GitHub的文档：MicroCity文档，也可以参考 MicroCtiy 在GitHub上的文档。

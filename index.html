# 开始使用MicroCityWeb

  1. 打开MicroCityWebopen in new window，默认加载方块旋转的示例代码。

  2. 点击运行按钮运行代码

  3. 程序运行，看到右侧视图中方块旋转。

如果想要深入了解MicroCityWeb的用法和使用技巧，可以参阅本站提供的 学习笔记

# 开始使用MicroCity

  1. 转到MicroCity的文档仓库下载 MicroCity的最新版本open in new window

  2. 打开 ScriptEditor 并输入:

Print("hello, world!")

  1. 将脚本另存为 test.mcs 并将其拖入 MicroCity

  2. 在程序主界面左侧转到 Modules 标签页，打开 test ，双击其中的 main ，执行刚刚编写的脚本。

  3. 程序界面下方的 Messages 栏中会输出执行脚本的信息。

更多MicroCity的使用方法可以参考本站搬运自GitHub的文档：MicroCity文档，也可以参考 MicroCtiyopen in new window 在GitHub上的文档。
